Purpose

This directory registers Zenodo records associated with the CRT research stack.

Zenodo records are treated as staged public research summaries.

They are:

* public checkpoints
* archival citation anchors
* release snapshots

They are not:

* peer-reviewed publications by default
* institutional endorsements
* final theorem completions
* full private R&D archives
