Purpose

This directory registers public release artifacts.

Artifacts may include:

* Zenodo records
* DOI-linked summaries
* public archival records
* release snapshots
* external public references

This directory does not automatically contain the full artifact files.

Default policy:

link first
copy later only if explicitly approved
