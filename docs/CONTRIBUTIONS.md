# Contributions (Structured)
## Purpose
This document defines contribution types without asserting proof completeness.
It is intended to guide paper writing and review alignment.
## Contribution categories
### 1. Definition-level contributions
Examples:
- New structural concepts
- New classification schemes
- New decomposition frameworks
Requirement:
- Clearly defined scope
- Separation from informal description
---
### 2. Structural contributions
Examples:
- Relationships between concepts
- Layered architecture
- Dependency structures
Requirement:
- Explicit assumptions
- No implicit equivalence claims
---
### 3. Diagnostic contributions
Examples:
- Failure mode classification
- Break-path reasoning
- Minimal violation analysis
Requirement:
- Clearly bounded applicability
- No universal claims unless proved
---
### 4. Conceptual unification
Examples:
- Linking determinization, canonicalization, and escape-resistance
- Relating multiple theoretical domains
Requirement:
- No overstatement of equivalence
- Explicit statement of limits
---
### 5. Methodological contributions
Examples:
- Audit-first development
- Claim-label discipline
- Separation of structure vs proof
Requirement:
- Clearly distinguished from technical results
## Non-contributions
The following are NOT considered contributions:
- Informal intuition without structure
- Unverified claims
- AI-generated arguments without validation
- Re-labeling existing ideas without new structure
## Warning
This document does not assert that any specific item is fully established.
All contributions must respect STATUS.md.
