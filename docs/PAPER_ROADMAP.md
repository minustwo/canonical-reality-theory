# Paper Roadmap
## Purpose
This document defines how the MST–CRT–IET research program may be decomposed into publishable units.
It does not introduce new claims.
## Principle
The repository is not a paper.
Papers must:
- isolate contributions
- control scope
- respect claim status
- avoid over-aggregation
## Proposed decomposition
### Paper 1 — Structure Theory / MST
Focus:
- Structure theory formalization
- Admissible interpretation families
- Determinization mechanisms
- Structural vs epistemic plurality
Role:
- Foundational layer
- Formal core
- Likely primary entry into academic space
---
### Paper 2 — Canonical Reality / CRT
Focus:
- Canonical reality framework
- Break-path analysis
- Canonical preservation vs failure
- Escape-style reasoning
Role:
- Extension of MST into robustness and canonical structure
- May require clearer formal boundary before submission
---
### Paper 3 — Institutional / IET layer
Focus:
- Institutional interpretation of structural invariants
- Escape cost and system-level behavior
- Application to AI, markets, or governance systems
Role:
- Applied or interdisciplinary layer
- May be field-dependent (CS / econ / systems)
## Non-goal
This roadmap does not:
- assert completeness
- assert correctness
- define final theorem sets
- guarantee publication outcomes
## Strategy
Recommended order:
1. MST (formal anchoring)
2. CRT (structural extension)
3. IET (application layer)
Each paper should stand independently.
