# Authorship and Privacy
## Purpose
This document defines the authorship, maintenance, institutional-status, and privacy boundary for this public repository.
## Project status
Canonical Reality Theory (CRT) is an independent personal research project.
This repository is a public AI-assisted research repository for the MST–CRT–IET architecture stack.
It is not:
- an institutionally endorsed project
- a university project
- an employer project
- a publisher project
- a conference or journal project
- a software product
- a finalized monograph
- a peer-reviewed publication archive unless explicitly stated

Privacy preference

The maintainer prefers not to expose unnecessary personal information in general repository documentation.

Repository contributors and AI agents should not add personal identifiers beyond what is already required by citation records, DOI records, or explicitly approved public artifacts.

Citation identity

Formal citation metadata may identify scholarly authorship separately from the GitHub account handle.

Citation records may include author information required for scholarly attribution.

General repository documentation should avoid duplicating personal identity unless needed for citation, authorship, or legal clarity.

Institutional non-endorsement

No claim in this repository should be interpreted as representing the position of any university, employer, publisher, conference, journal, AI provider, research institute, company, or other institution unless an artifact explicitly states otherwise.

AI assistance

AI systems were used as research assistants, drafting tools, adversarial auditors, coding agents, and organizational aids.

AI systems do not confer:

* authorship authority
* institutional endorsement
* peer-review status
* mathematical correctness

Final responsibility for accepted public artifacts and public release decisions remains with the human maintainer.

Peer-review boundary

Repository publication, Zenodo archival upload, DOI assignment, arXiv or SSRN availability, and AI-assisted internal audit are not substitutes for external peer review.

Unless explicitly stated otherwise, repository artifacts are not externally peer reviewed.

Default rule

If unsure, preserve maintainer privacy.

Do not add personal identifiers.

Do not imply institutional endorsement.

Do not imply peer review.
