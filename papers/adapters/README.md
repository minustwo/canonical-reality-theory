# CRT Language Adapter Stress Tests

This directory contains diagnostic stress-test summaries for Canonical Reality Theory
language adapters.

These files are **accepted diagnostic artifacts**, not sealed theorem files and not
peer-reviewed publications.

---

## Files

- **`CRT_LANGUAGE_ADAPTERS_SUMMARY_v0_2.md`** — Horizontal adapter stress tests
  across 16 language/system domains (~133 cuts). Tests whether each domain can
  supply the seven CRT adapter objects and at which layer the mapping passes or
  reaches an expected boundary.

- **`CRT_VERTICAL_STRESS_TEST_SUMMARY_v0_1.md`** — Vertical tests for four
  boundary/stability dimensions: CJC (Circular Justification Consistency),
  QNL (Quantum Non-Locality), adapter stability under composition/projection,
  and computational complexity (30 cuts).

---

## Status

Both files are diagnostic artifacts accepted within the CRT research program.
They show that across ~163 cuts, no unexpected failure mode was found.
All observed failures fell into known or explicitly named CRT boundary classes.

**These are not theorem papers.** They provide cross-domain diagnostic evidence
for the CRT adapter framework.

---

## Key results

- **16 domains tested**: Datalog, SQL, type theory, Prolog/ASP, control systems,
  DeFi, legal rules, LLM reasoning, non-monotonic logic, probabilistic logic,
  temporal logic, concurrent systems, CSP/MIP, NLP/MTT, category theory,
  quantum computing.

- **Two new boundary classes identified**:
  - **CJC** (Circular Justification Consistency): a new classical justification
    boundary, orthogonal to NAA/R0c.
  - **QNL** (Quantum Non-Locality): an extension boundary; classical CRT is
    intact, quantum entanglement requires non-local source semantics.

- **Zero unexpected failures** across all tested domains.
