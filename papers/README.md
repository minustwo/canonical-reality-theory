# Papers
## Purpose
This directory organizes potential paper tracks for the MST–CRT–IET research program.
These files are scaffolds only.
They are not:
- submitted papers
- peer-reviewed papers
- final manuscripts
- theorem repositories
- proof archives
## Structure
```text
papers/
├── mst/
├── crt/
└── iet/
```

Paper tracks

MST

Focus:

* structure theory
* admissible interpretations
* plurality
* determinization

Role:

foundational formal layer

⸻

CRT

Focus:

* escape-resistance

Role:

canonical preservation layer

⸻

IET

Focus:

* institutional interpretation
* invariant preservation
* applied systems
* domain-specific robustness

Role:

applied and interpretive layer

Reading rule

Read each track independently.

Do not collapse MST, CRT, and IET into one paper unless explicitly justified.

Claim discipline

All paper-track materials are governed by:

* docs/STATUS.md
* docs/CLAIM_LABELS.md
* docs/REVIEW_POLICY.md
* docs/RELEASE_POLICY.md

Warning

A scaffold is not a claim.

An outline is not a proof.

A positioning document is not validation.
