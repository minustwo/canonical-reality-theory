# Market Structure Theory (MST)

## Role in CRT

MST is the **structure layer** of Canonical Reality Theory.
It defines admissible worlds and structural canonicality.

CRT takes MST worlds as inputs and asks:
when are they justified, robust, stable, and executable?

## Core objects

| Object | Definition |
|---|---|
| $\mathcal{P}_T(\Pi)$ | Admissible interpretation set |
| $D_{\mathrm{comp}}$ | Canonical completion operator |
| $\mathrm{SCSet}_T(\Pi)$ | Set of structurally canonical worlds |
| $V_T(x)$ | Validity predicate |

## Layers

- **MST Layer 1** (structure): which worlds are admissible?
- **MST Layer 2 / CRT Layer 2** (justification): why is a world admissible?
  Defines $\mathcal{J}_{\mathrm{declared}} \to \mathcal{J}_{\mathrm{valid}}$ pipeline.
  This is the direct input to CRT Bridge conditions C1–C2.

## Key result relevant to CRT

MST Layer 2 establishes that Layer-1 uniqueness
($|\mathcal{P}_T(\Pi)| = 1$) does not imply Layer-2 justification
uniqueness or full Layer-2 assignment. This is the structural
motivation for CRT Bridge conditions C1–C4.
