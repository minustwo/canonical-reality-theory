# Contributing
## Status
This repository is not currently open for external contributions.
## Reason
This repository is part of an active research program with:
- Private development branches
- Ongoing audits
- Controlled public release boundaries
Uncontrolled contributions may:
- Break claim discipline
- Introduce theorem promotion
- Mix private and public material
- Violate audit structure
## Future possibility
Contribution policy may be defined later.
Possible future contribution modes:
- Issue-based discussion
- External review feedback
- Structured audit contributions
- Controlled pull requests
## Rule
Until explicitly opened:
> Do not submit pull requests.
