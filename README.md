# Canonical Reality Theory
**Status:** Public initialization repository  
**Review status:** AI-assisted research; not peer-reviewed  
**Scope:** Curated public documentation for the MST–CRT–IET architecture stack
This repository hosts the public-facing structure for **Canonical Reality Theory (CRT)** and its related architecture stack:
- **MST** — Market Structure Theory / Structure Theory determinization layer
- **CRT** — Canonical Reality Theory / canonical reality and break-path layer
- **IET** — Institutional / Invariant / Interpretive Escape Theory layer
This public repository is intentionally curated. Raw research notes, exploratory drafts, private audits, failed branches, and unreviewed theorem experiments are not included here unless explicitly approved for public release.
## Repository principles
1. **No theorem promotion**  
   Conditional, conjectural, partial, sketched, or audit-local results must remain labeled as such.
2. **No proof compression**  
   Public documentation may describe architecture and status, but must not silently summarize, replace, or strengthen proof content.
3. **No private-source leakage**  
   Private repository material may only be imported after explicit human approval.
4. **English only**  
   Public-facing files in this repository must be written in English.
5. **AI-assisted disclosure**  
   This repository may contain AI-assisted research writing, structure planning, review notes, and documentation. Such material is not peer review.
