\section{Expressivity and Embedding Framework}
\label{sec:expressivity}

The minimal language $(\mathrm{At},F,\mathcal{B},\mathcal{A})$ can faithfully embed
existing frameworks from logic, security, and dynamics.

\begin{definition}[Encoding interface]
A faithful encoding $\mathrm{Enc}(X)$ satisfies:
(F1)~state fidelity; (F2)~structural fidelity; (F3)~dynamic fidelity.
\end{definition}

\begin{proposition}[Canonical embeddings]
Class I systems (Datalog, TMS): closed $F$, empty $\mathcal{B}$.
Class II systems (blockchain): certified external $\mathcal{B}$.
Class III systems (DeFi, ICT): unavoidable $\mathrm{UES}$, finite $C_{\min}$.
\end{proposition}
