# Code of Conduct
## Purpose
This repository is a research environment, not a community forum.
## Expected behavior
- Stay focused on content, structure, and claims
- Avoid speculative reinterpretation of results
- Respect claim labels and boundaries
- Do not promote unproven statements
## Prohibited behavior
- Misrepresenting claim status
- Treating drafts as established theory
- Extracting private or unpublished material
- Using repository content for misleading claims
## Enforcement
The maintainer may:
- ignore contributions
- remove content
- restrict interaction
This repository is not obligated to maintain open participation.
